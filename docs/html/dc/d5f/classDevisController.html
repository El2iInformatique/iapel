Description détaillée

Contrôleur responsable de la gestion et de la suppression des devis signés.

Ce contrôleur permet :

  • De vérifier l’existence d’un devis (PDF ou version certifiée).
  • De supprimer un devis ainsi que son token associé de manière sécurisée.
  • De protéger les accès via un système de jeton secret d’organisation.

Documentation des fonctions membres

◆ check()

App::Http::Controllers::DevisController::check ( class Request $request,
$noToken )

Vérifie la présence d’un devis associé à un token donné.

Cette méthode :

  • Vérifie l’existence d’un token valide.
  • Contrôle le secret token d’accès pour s’assurer de l’autorisation.
  • Confirme si un fichier PDF ou PDF certifié existe pour ce devis.
Paramètres
Request$requestRequête HTTP contenant le header secret-token.
string$noTokenToken public du devis à vérifier.
Renvoie
JsonResponse Réponse JSON indiquant si le devis existe.
Exceptions

Illuminate\Http\Exceptions\HttpResponseException Si le token n’est pas valide ou si l’utilisateur n’est pas autorisé.

Exceptions

Exception En cas d’erreur lors de la vérification.

Note
Cette méthode ne renvoie jamais le contenu du fichier, uniquement son existence.

◆ delete()

App::Http::Controllers::DevisController::delete ( class Request $request,
$noToken )

Supprime un devis et les fichiers associés à partir d’un token.

Cette méthode :

  • Valide le token fourni dans l’URL.
  • Vérifie le secret token d’autorisation (client ou administrateur).
  • Supprime le fichier PDF du devis, le PDF certifié, le répertoire du devis ainsi que le token associé dans la base de données.
Paramètres
Request$requestRequête HTTP contenant le header secret-token.
string$noTokenToken public du devis à supprimer.
Renvoie
JsonResponse Réponse JSON indiquant le succès ou l’échec de l’opération.
Exceptions

Illuminate\Http\Exceptions\HttpResponseException Si le token n’est pas valide ou si l’utilisateur n’est pas autorisé.

Exceptions

Exception En cas d’erreur lors de la suppression des fichiers.

Note
Cette opération est irréversible. Les fichiers supprimés ne peuvent pas être restaurés.
