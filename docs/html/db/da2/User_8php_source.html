10/**
11 * @class User

13 *
14 * Ce modèle hérite de la classe Authenticatable de Laravel et gère tous les aspects

18 * Fonctionnalités principales :

26 * @package App\Models

33 * @warning Ne jamais exposer les mots de passe ou tokens dans les réponses API.

35 * @example
36 * ```php

38 * $user = User::create([

44 * // Authentification

46 * // Utilisateur connecté

116 /**

119 * Méthode utilitaire pour vérifier rapidement si un utilisateur

124 * @example
