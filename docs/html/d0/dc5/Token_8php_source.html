10 * @class Token
11 * @brief Modèle de gestion des tokens de signature électronique pour les devis.
12 *
13 * Ce modèle centralise la gestion des tokens sécurisés permettant la signature électronique

15 * du document de signature et au positionnement des éléments graphiques (signatures, dates).
16 *
17 * Fonctionnalités principales :
18 * - Génération de tokens uniques avec durée de validité limitée.
19 * - Stockage des informations financières du devis (HT, TVA, TTC).
20 * - Gestion des coordonnées de positionnement pour signatures et dates.
21 * - Liaison avec l'organisation et le client pour traçabilité.
22 * - Support de l'expiration automatique des tokens.

24 *
25 * @package App\Models
26 * @version 2.0

30 * @note Utilise la table "tokens" en base de données avec gestion des timestamps Laravel.

96 /**
97 * @brief Génère un nouveau token de signature pour un devis.
98 *
99 * Cette méthode statique crée un token sécurisé unique avec toutes les informations
100 * nécessaires pour permettre la signature électronique d'un devis. Le token contient
101 * les données financières, les coordonnées de positionnement des signatures et dates,
102 * ainsi que les informations de traçabilité.
103 *
104 * Le token généré permet :
105 * - L'accès sécurisé au document de signature via une URL unique.
106 * - L'affichage des informations du devis au client.
107 * - Le positionnement précis des signatures et dates sur le PDF.
108 * - La traçabilité complète de la signature (qui, quand, quoi).
109 * - L'expiration automatique après un mois.
110 *
111 * @param int $organisationId Identifiant de l'organisation émettrice du devis.
112 * @param string $tiers Nom ou raison sociale du client/tiers signataire.
113 * @param int $devisId Identifiant unique du devis dans le système.
114 * @param string $clientEmail Adresse email du client pour notifications.
115 * @param string $titre Titre ou description courte du devis.
116 * @param float $montantHT Montant hors taxes en euros.
117 * @param float $montantTVA Montant de la TVA en euros.
118 * @param float $montantTTC Montant toutes taxes comprises en euros.
119 * @param array $coords Coordonnées de positionnement des éléments de signature.
120 * @param int $nbpages Nombre de pages du document PDF à signer.
121 *
122 * @return Token Instance du token créé avec toutes les données.
123 *
124 * @throws \Illuminate\Database\QueryException Si la création en base échoue.
125 *
126 * @example
127 * ```php
128 * // Coordonnées pour un document A4 (595x842 points PDF)
129 * $coords = [
130 * 'x_signature' => 150, // Position X de la signature client
131 * 'y_signature' => 100, // Position Y de la signature client
132 * 'x_date' => 400, // Position X de la date de signature
133 * 'y_date' => 100 // Position Y de la date de signature
134 * ];
135 *
136 * $token = Token::generateToken(
137 * 1, // ID organisation
138 * 'Entreprise Martin SARL', // Nom du client
139 * 456, // ID du devis
140 * 'martin@entreprise.com', // Email client
141 * 'Installation électrique', // Titre du devis
142 * 2500.00, // Montant HT
143 * 500.00, // Montant TVA (20%)
144 * 3000.00, // Montant TTC
145 * $coords, // Coordonnées
146 * 2 // Nombre de pages
147 * );
148 *
149 * echo "Token généré : " . $token->token;
150 * echo "Expire le : " . $token->expires_at;
151 * ```
152 *
153 * @example
154 * ```php
155 * // Génération pour un petit devis simple
156 * $coords = [
157 * 'x_signature' => 100,
158 * 'y_signature' => 50,
159 * 'x_date' => 300,
160 * 'y_date' => 50
161 * ];
162 *
163 * $token = Token::generateToken(
164 * 2, 'Particulier Dupont', 789,
165 * 'dupont@email.com', 'Réparation urgente',
166 * 150.00, 30.00, 180.00, $coords, 1
167 * );
168 * ```
169 *
170 * @note Le token généré fait 40 caractères aléatoires pour sécurité maximale.
171 * @note L'expiration est fixée à un mois après la création du token.
172 * @note Les coordonnées doivent être en points PDF (1 point = 1/72 pouce).
173 * @warning Vérifier la cohérence des montants (HT + TVA = TTC) avant appel.
174 * @warning S'assurer que les coordonnées sont dans les limites du document.
175 * @see Str::random() Pour la génération du token aléatoire sécurisé.
176 */

198 /**
199 * @brief Vérifie si le token est encore valide.
200 *
201 * Contrôle que le token n'a pas expiré et n'a pas encore été utilisé
202 * pour une signature. Un token expiré ou déjà utilisé ne peut plus
203 * servir à signer un document.
204 *
205 * @return bool True si le token est valide et utilisable, false sinon.
206 *
207 * @example
208 * ```php
209 * $token = Token::where('token', $tokenString)->first();
210 *
211 * if ($token && $token->isValid()) {
212 * // Permettre l'accès à la signature
213 * return view('signature', compact('token'));
214 * } else {
215 * // Token expiré ou déjà utilisé
216 * return redirect()->route('error')->with('message', 'Token invalide');
217 * }
218 * ```
219 */
220 public function isValid(): bool
221 {
222 return !$this->used && $this->expires_at > now();
223 }

250 /**
251 * @brief Vérifie si le token va expirer bientôt.
252 *
253 * Détermine si le token expire dans les prochaines 24 heures,

255 *
256 * @return bool True si le token expire dans moins de 24h, false sinon.

260 * // Système de rappels automatiques

271 public function isExpiringSoon(): bool
272 {
273 return $this->expires_at <= now()->addDay() && $this->expires_at > now();
274 }

276 /**
277 * @brief Calcule le montant total avec vérification de cohérence.
278 *
279 * Vérifie que la somme HT + TVA correspond bien au montant TTC
280 * et retourne le montant TTC avec une marge d'erreur acceptable.
281 *
282 * @return float Montant TTC vérifié.
283 * @throws \InvalidArgumentException Si les montants sont incohérents.
284 *
285 * @example
286 * ```php
287 * try {
288 * $montantVerifie = $token->getVerifiedTotal();
289 * echo "Montant TTC vérifié : " . number_format($montantVerifie, 2) . " €";
290 * } catch (\InvalidArgumentException $e) {
291 * Log::error("Incohérence montants token {$token->id}: " . $e->getMessage());
292 * }
293 * ```
294 */
295 public function getVerifiedTotal(): float
296 {
297 $calculatedTTC = $this->montant_HT + $this->montant_TVA;
298 $difference = abs($calculatedTTC - $this->montant_TTC);
299
300 // Tolérance d'erreur de 0.01€ pour les arrondis
301 if ($difference > 0.01) {
302 throw new \InvalidArgumentException(
303 "Incohérence montants: HT({$this->montant_HT}) + TVA({$this->montant_TVA}) ≠ TTC({$this->montant_TTC})"
304 );
305 }
306
307 return $this->montant_TTC;
308 }
